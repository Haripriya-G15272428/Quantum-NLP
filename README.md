# Quantum NLP Project

A machine learning / NLP based project that analyzes text data using Natural Language Processing techniques.

## 🚀 Features
- Text preprocessing
- NLP based classification / prediction
- User-friendly interface
- Model integration using Python
- Fast and accurate results

## 🛠️ Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- NLP

## 📂 Project Structure
project-folder/
│── app.py
│── model.pkl
│── vectorizer.pkl
│── requirements.txt
│── templates/
│── static/

## ▶️ How to Run

1. Clone repository

git clone https://github.com/Haripriya-G15272428/Quantum-NLP.git

2. Install dependencies

pip install -r requirements.txt

3. Run application

python app.py
